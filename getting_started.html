1. Install the plugin

Search for "Nextflow" in Settings → Plugins → Marketplace and click Install. The LSP4IJ dependency is installed automatically.

2. Verify Java 17+

The Nextflow Language Server requires Java 17 or later. Confirm with java -version in your terminal.

3. Open a Nextflow project

Open a directory containing .nf files. The language server starts automatically and provides diagnostics, completion, hover, and navigation.

4. Preview workflow DAGs

Click the Preview DAG code lens above any workflow block to render an interactive graph.
