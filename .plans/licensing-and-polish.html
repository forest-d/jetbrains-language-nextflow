Licensing and Polish

Context

Prepare the Nextflow JetBrains plugin for public distribution on the JetBrains Marketplace as MIT-licensed free software. This covers four areas: licensing/attribution, marketplace metadata, LLM-style artifact removal, and code quality cleanup.

Licensing landscape

  • Nextflow Language Server (Apache 2.0) - downloaded at runtime, not bundled. Apache 2.0 is compatible with MIT; attribution required.
  • VS Code Nextflow Extension (MIT) - TextMate grammars copied from this repo into src/main/resources/textmate/. MIT-to-MIT is straightforward; attribution required.
  • LSP4IJ (EPL 2.0) - installed as a separate plugin dependency, not bundled.

Marketplace best practices

  • Plugin name should be 1-4 words, no "Plugin" or "Support" suffix.
  • Vendor should reflect the actual author, not a project name.
  • Description first 40 characters generate preview cards.
  • No em dashes, marketing language, or LLM-style phrasing.
  • Plugin icons required (already present).
